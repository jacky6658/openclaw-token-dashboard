## 🔧 自訂調整

### 調整霓虹強度
如果覺得太刺眼，可降低發光強度：

```css
/* 找到 box-shadow 屬性，降低 alpha 值 */
box-shadow: 
  0 0 30px rgba(0, 240, 255, 0.3),  /* 改為 0.2 */
  0 8px 25px rgba(0, 240, 255, 0.2); /* 改為 0.1 */
```

### 調整背景網格
如果覺得網格太明顯：

```css
/* body::before 中調整透明度 */
background-image: 
  linear-gradient(rgba(0, 240, 255, 0.01) 1px, transparent 1px), /* 0.03 → 0.01 */
  linear-gradient(90deg, rgba(0, 240, 255, 0.01) 1px, transparent 1px);
```

### 調整動畫速度
如果覺得動畫太快/太慢：

```css
/* 調整動畫時長 */
animation: neon-flow 5s linear infinite; /* 3s → 5s */
animation: pulse-glow 12s ease-in-out infinite; /* 8s → 12s */
```
