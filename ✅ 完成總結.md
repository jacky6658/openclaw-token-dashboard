```html

```

### 3. WebRTC IP 偵測
```javascript
// 自動偵測本機 IP
const pc = new RTCPeerConnection();
// 無需手動輸入
```

### 4. 一鍵打包
```bash
# 自動排除不需要的檔案
zip -r output.zip project \
  -x "node_modules/*" \
  -x "*.db"
```
