# 📦 分享給朋友使用指南

> 如何將 Dashboard 打包分享給其他人使用

---

## 🎯 打包步驟

### 方法一：壓縮專案（推薦）

```bash
# 1. 進入專案父目錄
cd /Users/user/clawd/projects

# 2. 壓縮整個專案（不包含 node_modules）
zip -r token-dashboard.zip token-dashboard \
  -x "token-dashboard/node_modules/*" \
  -x "token-dashboard/.git/*" \
  -x "token-dashboard/backend/db/*"

# 3. 壓縮檔會產生在：
# /Users/user/clawd/projects/token-dashboard.zip
```

### 方法二：使用桌面（圖形化）

1. **複製專案到桌面**
   ```bash
   cp -r /Users/user/clawd/projects/token-dashboard ~/Desktop/token-dashboard-分享版
   ```

2. **清理不需要的檔案**
   ```bash
   cd ~/Desktop/token-dashboard-分享版
   rm -rf node_modules
   rm -rf .git
   rm -rf backend/db/*.db
   ```

3. **壓縮**
   - 在 Finder 中右鍵點擊資料夾
   - 選擇「壓縮」
   - 產生 `token-dashboard-分享版.zip`

## 👥 給朋友的使用說明

### 系統需求
- ✅ macOS / Linux / Windows
- ✅ 已安裝 Node.js (>= 14.0.0)
- ✅ 已安裝並配置 OpenClaw

### 啟動方式（3 種）

#### 🎯 方法一：雙擊啟動（最簡單）

1. **解壓縮 `token-dashboard.zip`**

2. **雙擊「啟動 Dashboard.command」檔案**
   - macOS 首次開啟可能需要：
     - 右鍵 > 打開 > 確認
   - 會自動：
     - ✅ 檢查環境
     - ✅ 安裝依賴
     - ✅ 啟動 Gateway
     - ✅ 開啟瀏覽器

3. **完成！** 🎉

#### 💻 方法二：終端機啟動

```bash
# 1. 進入專案目錄
cd /path/to/token-dashboard

# 2. 安裝依賴（首次使用）
npm install

# 3. 啟動
./start.sh

# 或直接執行
node backend/server.js
```

#### 📱 方法三：開啟手機訪問指南

1. **雙擊「手機訪問指南.html」**
2. **點擊「自動偵測我的 IP」**
3. **用手機掃描 QR Code**
4. **完成！**

## 📱 手機訪問說明

### 快速步驟

1. **確保電腦和手機在同一個 WiFi**

2. **取得電腦 IP 位址**
   - 方法 A：開啟「手機訪問指南.html」自動偵測
   - 方法 B：雙擊「啟動 Dashboard.command」會顯示

3. **在手機瀏覽器輸入**
   ```
   http://您的IP:3737
   例如：http://192.168.1.100:3737
   ```

### 詳細步驟

#### 1️⃣ 連接同一個 WiFi

**確認方式：**
- 電腦：系統偏好設定 > 網路 > WiFi
- 手機：設定 > WiFi
- 兩者顯示的 WiFi 名稱必須相同

#### 2️⃣ 查詢電腦 IP

**macOS 圖形化方式：**
1. 系統偏好設定
2. 網路
3. WiFi（已連接）
4. 右側顯示「IP 位址：xxx.xxx.xxx.xxx」

**或在終端機執行：**
```bash
ifconfig | grep "inet " | grep -v 127.0.0.1
```

**或雙擊啟動器時會顯示：**
```
📱 手機訪問（同 WiFi）：
   http://192.168.1.100:3737
```

#### 3️⃣ 手機訪問

**在手機瀏覽器（Safari/Chrome）輸入：**
```
http://[您的IP]:3737
```

**例如：**
```
http://192.168.1.100:3737
```

## 🔒 安全性說明

### ✅ 安全使用
- 只在私人 WiFi 網路中使用
- 數據不會傳到網際網路
- 只有同 WiFi 的設備能訪問
- 所有數據都在本機

### ⚠️ 注意事項
- ❌ 不要在公共 WiFi（咖啡廳）中使用
- ❌ 不要分享您的 IP 給陌生人
- ❌ 不要在不信任的網路中開啟

## 🐛 常見問題

### Q: 手機無法連接？

**A: 檢查以下項目**

1. **WiFi 檢查**
   ```
   ✅ 電腦和手機在同一個 WiFi
   ✅ 不是「同名稱」而是「同一個網路」
   ```

2. **Dashboard 檢查**
   ```
   ✅ 電腦上的 Dashboard 正在運行
   ✅ 電腦瀏覽器能訪問 http://localhost:3737
   ```

3. **防火牆檢查**
   ```
   系統偏好設定 > 安全性與隱私 > 防火牆
   允許 Node.js 接收連線
   ```

4. **端口檢查**
   ```bash
   # 確認 3737 端口正在監聽
   lsof -i :3737
   ```

### Q: 啟動器無法執行？

**A: macOS 安全性設定**

```
首次使用需要：
1. 右鍵點擊「啟動 Dashboard.command」
2. 選擇「打開」
3. 點擊「打開」確認
4. 之後就能直接雙擊使用
```

### Q: 提示「未安裝 Node.js」？

**A: 安裝 Node.js**

```
1. 前往 https://nodejs.org/
2. 下載 LTS 版本
3. 安裝後重啟終端機
4. 驗證：node -v
```

### Q: 朋友沒有安裝 OpenClaw？

**A: Dashboard 仍可使用**

```
✅ Dashboard 可以獨立運行
⚠️ 但以下功能無法使用：
   - 切換模型（需要 OpenClaw）
   - 即時配額監控（需要 OpenClaw）
   - Gateway 狀態（需要 OpenClaw）

✅ 以下功能仍可用：
   - 查看歷史數據（如果有資料庫）
   - 介面展示
```

分享給朋友的壓縮包應包含：

```
token-dashboard/

├── start.sh                   ✅ 終端機啟動
├── package.json               ✅ 依賴清單
├── frontend/                  ✅ 前端檔案
├── backend/                   ✅ 後端檔案
└── *.md                       ✅ 其他文檔
```

**不需要包含：**
```
❌ node_modules/  （太大，會自動安裝）

```
